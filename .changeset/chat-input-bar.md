[LG-4121](https://jira.mongodb.org/browse/LG-4121): `InputBar` renders results menu in top layer using popover API. As a result, the following props are deprecated and removed:
- `portalClassName`
- `portalContainer`
- `portalRef`
- `scrollContainer`
- `usePortal`

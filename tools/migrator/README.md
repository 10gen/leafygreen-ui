# Migrator

![npm (scoped)](https://img.shields.io/npm/v/@leafygreen-ui/migrator.svg)

## Installation

### Yarn

```shell
yarn add @lg-tools/migrator
```

### NPM

```shell
npm install @lg-tools/migrator
```

## Usage

`yarn lg migrator <migration> <path>`
